# BLOGGING-PLATFORM

 This is a simple blogging website built with MERN. Users can write posts, edit them, and read blogs in a clean and responsive UI. 
 The project shows how to use components, routing, and React hooks to build a small blog application.

# Tech Stack

 1. MongoDB
 2. Express
 3. React
 4. Node.js
 5.TailwindCss
